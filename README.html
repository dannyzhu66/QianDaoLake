旅行贴士

门票预订

提前在"千岛湖旅游官方平台"公众号预订船票和景点门票，旺季周末务必提前购票。

住宿推荐

选择中心湖区（绿城度假区、秀水街附近）或东南湖区码头附近的酒店，交通便利。

防晒防暑

夏季千岛湖紫外线强，必备帽子、墨镜、防晒霜、驱蚊液，多喝水预防中暑。

餐饮提示

吃鱼头汤问清价格（通常按斤卖），选择明码标价的店家，推荐秀水街区域餐厅。

必备物品

舒适步行鞋、泳衣（皮划艇用）、毛巾、手机防水袋、充电宝、常用药品。

千岛湖位置


千岛湖位于浙江省杭州市淳安县
距离上海约300公里

推荐酒店

绿城度假酒店

中心湖区核心位置，湖景房视野绝佳，设施完善，亲子友好

千岛湖开元颐居

性价比高，靠近东南湖区码头，现代简约风格

千岛湖伯瑞特度假酒店
